# RunLab Calculator

Modern running calculator with two features:

1. Speed converter: `km/h` -> `min/km`.
2. Distance time estimator: `distance (0-1000m)` + `pace (min/km)` -> expected time.

## Local run

Open `index.html` in your browser.

## Example

- `10 km/h` => `6:00 min/km`
- `330m` at `4:00 min/km` => `1:19`
